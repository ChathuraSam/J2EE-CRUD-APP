# J2EE-CRUD-APP
Simple project done by using JSP, Servlets that handles CRUD operations.

Tools and technologies Used
----------------------------
- JSP 2.2+
- JDK 1.8 or later
- Apache tomcat 8.5
- JSTL 1.2.1
- Servlet API 2.5
- mysql-connector-java-8.0.13
